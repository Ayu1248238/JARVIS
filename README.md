# JARVIS---an-advance-virtual-voice-assistant-that-help-you-with-your-stuff
JARVIS is an advanced AI desktop assistant that uses LangChain ReAct, Google Gemini, and LiveKit for real-time voice interaction, reasoning, and automation. It can open apps, control the system, search, type, move the mouse, and perform intelligent tasks with a futuristic JARVIS-like interface.
